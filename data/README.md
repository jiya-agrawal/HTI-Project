# Data Directory

This directory stores all experimental data collected during the study.

## Files Generated:

### results.csv
Main data export containing all metrics for each puzzle completion.

### interactions.json
Detailed log of every participant interaction with timestamps.

### summary.json
Aggregated statistics across all participants.

---

**Note:** Data files are automatically created when the first participant completes a puzzle.

All files are in `.gitignore` to protect participant privacy.
